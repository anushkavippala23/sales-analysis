# 📊 E-Commerce Sales Analysis & Prediction

## Project Description
This project analyzes E-Commerce Sales Data to understand sales trends and predict future sales using Machine Learning.

## 🔍 What This Project Does
- **Sales Trends** — Monthly, Quarterly revenue patterns
- **Regional Analysis** — Region wise sales comparison
- **Top Products** — Best selling products identification
- **ABC Analysis** — Product classification (High/Medium/Low)
- **ML Prediction** — Linear Regression & Random Forest models

## 🛠️ Technologies Used
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- Streamlit

## 📁 Dataset
E-Commerce sales data with Order Date, Product Name, Region, Sales, Quantity, Profit columns.
